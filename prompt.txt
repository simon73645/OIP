Ich habe folgende Probleme:
1. Speichern
Meine Simulatin enthält die Möglichkeit eine Simulation zu speichern.
Jedoch wird nur die Position der Objekte gespeichert gespeichert. Wird ein ConveyorBelt beispielsweise
mit der "Scale" Methode verlängert oder verbreitert, so wird dies nicht gespeichert. Läd man die Szene,
so wird der Belt Convey an seiner Ursprünglichen Position zwar platziert, die Verbreiterung oder
Verlängerung wird jedoch nicht gespeichert. Bitte passe das ganze so an, dass dies auch gespeichert wird, sodass
sich die Szene komplett laden lässst. Attribute wie in Problem 2 (Farbe) müssen natürlich auch gespeichert werden.

2. Box Farben
Es gibt einen Color Sensor. Bitte füge hinzu, dass jede Box entweder rot, blau oder grün sein kann.
Dies soll man aus einem extra Menü auswählen können, sodass der Color Sensor später die Farbe der Boxen unterscheiden kann.
Sorge dafür, dass das ganze implementiert wird und füge dem tutorial im Ordner tutorial eine detallierte Anleitung hinzu.
